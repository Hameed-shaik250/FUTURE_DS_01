## Task 1 – Sales Dashboard

Tools Used:
Power BI

Description:
Created an interactive Sales Dashboard to analyze business performance.

Dashboard includes:
- Total Sales KPI
- Sales by Category
- Sales by Region
- Monthly Sales Trend
- Top 10 Products by Sales

This project demonstrates data visualization and business insight generation using Power BI.
